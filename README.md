## Building and running

### Instalación

```bash
pip install -r requirements.txt
```

### Configuración

Necesitas crear el archivo **.env** en la raíz del proyecto con las variables de entorno.

### Ejecutar

```bash
uvicorn main:app --host 0.0.0.0 --port 3000
```
